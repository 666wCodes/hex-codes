# hex-codes

> Easily use hex codes in your project

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i hex-codes
```

## Requiring the package

```js
const hex = require('hex-codes')
```
### Using the hex codes
```javascript
const hex = require('hex-codes')
const variableName = hex.blue()
console.log(variableName)
```
